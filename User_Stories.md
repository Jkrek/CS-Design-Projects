# DraftVision User Stories

**Project Title:** DraftVision – Machine Learning Powered NFL Prospect Success Prediction Platform  
**Team Members:**  
- Jared Krekeler

---

## User Stories

1. As an NFL scout, I want to enter the name of a college football player and view a probability score of their projected NFL success so that I can quickly prioritize which prospects to focus my in-depth evaluation on.

2. As a general manager, I want to compare success probabilities for multiple prospects on my draft board so that I can make more data-driven decisions when choosing between players with similar grades.

3. As a data analyst for a football organization, I want to see which features (such as production metrics, athletic testing, and school context) most influence the model’s prediction so that I can better explain and justify the recommendations to coaches and decision makers.

4. As a draft content creator(me), I want to generate a simple visual summary of a player’s success probability and key strengths so that I can include clear, data-driven graphics in my articles and videos.

5. As a passionate football fan, I want to look up prospects and view their projected NFL success so that I can better understand my team’s draft choices and follow my favorite players into the league.
